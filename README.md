# AI LLM Engineer Handbook

A comprehensive open-source handbook covering Mathematics, Machine Learning, Deep Learning, Transformers, LLMs, RAG, AI Agents, MLOps and Rust.

## Bắt đầu từ đâu?

Xem [00-README/00-README.md](00-README/00-README.md) để có tổng quan toàn bộ sách và mục lục 21 chương, hoặc [01-Roadmap/01-Roadmap.md](01-Roadmap/01-Roadmap.md) để bắt đầu lộ trình học.

## Cấu trúc

Sách được tổ chức thành 1 thư mục cho mỗi chương (đánh số `00` đến `21`, ví dụ `04-Transformer/`), mỗi thư mục chứa file nội dung `.md` và notebook `.ipynb` cùng tên (trừ chương `13-Rust-AI`). Xem [SUMMARY.md](SUMMARY.md) để có danh sách đầy đủ.
